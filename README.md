# ddnsto 是什么？
* 可以配置一个域名，远程到家里的私有网络
* 可以用网页，远程家里的电脑桌面
* 可以安装 EasyExplorer 与 APP，备份手机的数据到家里
* 用 EasyExplorer　对应的 APP 看家里的视频，不管有没有公网 IP

官网：https://www.ddnsto.com
旧：https://www.ddns.to 暂时不用

有问题请在[这里反馈](https://github.com/koolshare/ddnsto/issues/new)。
