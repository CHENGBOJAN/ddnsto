# EasyExplorer 解决了什么痛点？
* APP 备份文件到家里，走 P2P 省流量
* APP 看家里存的视频
* 增量备份，只传改动的数据
* 全平台相互管理与备份，一个都不能少（路由器，NAS，手机，电脑)
